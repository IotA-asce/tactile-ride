# V0.1 Electronics Prototype Sources

`TactileRide_V0.1_Wiring.svg` is a draft logical wiring concept for a future
five-input bench experiment. Its D0 through D4 labels are not selected-board
pins, firmware configuration, a reviewed schematic, or a permission to connect
a battery.

Using this source requires board selection, component selection, protection and
charging review, a safe bench procedure, and a firmware input decision. See the
[V0.1 blueprint index](../../../docs/blueprints/v0.1/README.md) before using
this source.
